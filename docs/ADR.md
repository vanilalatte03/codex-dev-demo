# Architecture Decision Records

## 운영 규칙
- ADR 1~3개는 이 파일에 기록한다.
- 4번째 ADR이 생기면 `docs/adr/0001-title.md` 형식으로 분리한다.
- 분리 후 이 파일은 ADR 인덱스 역할만 유지한다.
- ADR은 결정을 바꾸기 전에 먼저 추가하거나 갱신한다.
